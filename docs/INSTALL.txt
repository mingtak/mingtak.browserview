mingtak.browserview Installation
--------------------------------

To install mingtak.browserview using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``mingtak.browserview`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        mingtak.browserview
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
